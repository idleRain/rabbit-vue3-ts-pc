# About
> 这是一个基于 Vue3 + TypeScript 的电商练习项目
>
> 相关技术点 : Vue3 + TypeScript + VueRouter + pinia + axios + less

## 运行
- `yarn` 或 `npm install`

- `yarn dev` 或 `npm run dev`
